# to-do list
sample to-do list in python

## Features
-simple gui  
-add task  
-remove task  
-change task  
-list task  
-mark tasks  
-saving/loading csv  

## How to use
go to to-do-list directory  
```
cd C:\your\path\to\directory\to-do-list>
```
run script  
```
python src\main.py
```
